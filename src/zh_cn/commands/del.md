# del key

Delete specified key.

## 参数

* `key` - 

## 返回值

Status reply. You can not determine whether the key exists or not by delete command.

## 示例

	ssdb 127.0.0.1:8888> set a 123
	ok
	(0.000 sec)
	ssdb 127.0.0.1:8888> del a
	ok
	(0.000 sec)
	ssdb 127.0.0.1:8888> del a
	ok
	(0.000 sec)
	ssdb 127.0.0.1:8888>
